# LineaLabs Catalog System

Web-based catalog management system for LineaLabs, built with PHP, MySQL and Bootstrap.

## Description

This project is a web system developed for managing products in a custom MDF catalog.
It is being developed for academic purposes and real use in the LineaLabs business.

## Technologies (planned)

- PHP
- MySQL
- Bootstrap
- HTML
- CSS
- JavaScript

## Features (planned)

- Product catalog
- Admin panel
- Authentication
- CRUD operations
- Database integration
